# BitTrade API Catalog (Official Docs, JP)

このディレクトリは BitTrade 公式ドキュメントに限定した API カタログです。

## Files
- `sources.md`: 公式到達ページの証跡一覧
- `rest-api.md`: REST API 一覧（Public / Private / Other）
- `websocket.md`: WebSocket 一覧（Public / Private）
- `data.md`: データ配布系（公式記載有無）
- `catalog.json`: 機械可読SSOT
- `templates.md`: 次回更新テンプレ
- `CHANGELOG.md`: チェック履歴（append-only）

## Scope note
- 公式サイト導線および `api-doc.bittrade.co.jp` 掲載内容のみを対象としています。
