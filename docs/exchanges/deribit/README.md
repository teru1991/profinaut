# Deribit Official Docs Catalog (SSOT)

This directory contains an **official-docs-only** catalog for Deribit:
- JSON-RPC over WebSocket (RPC + subscriptions)
- JSON-RPC over HTTP
- FIX API
- Official Releases / Changes diffs

Completeness evidence and crawl roots are in `sources.md`.

## Files
- sources.md: SSOT evidence of coverage
- jsonrpc-ws.md: JSON-RPC over WebSocket catalog (methods + protocol)
- jsonrpc-http.md: JSON-RPC over HTTP catalog (methods)
- subscriptions.md: public/subscribe & private/subscribe channels catalog
- fix.md: FIX catalog (official)
- data.md: Official data distribution catalog (if documented)
- diffs.md: Official releases/changes diffs (official only)
- catalog.json: Machine-readable SSOT (1:1 with markdown tables)
- templates.md: Update templates
- CHANGELOG.md: Append-only change history
