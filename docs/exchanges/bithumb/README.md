# Bithumb Official Docs Catalog (SSOT)

This directory contains an **official-docs-only** catalog for Bithumb:
- REST: documented public/private endpoints found from the official reference navigation
- WebSocket: documented public/private channels + WS protocol/common pages
- Official diffs extracted from Bithumb changelog pages

Completeness evidence is tracked in `sources.md`.

## Files
- sources.md: SSOT evidence of coverage and crawl roots
- rest-api.md: REST endpoints catalog (public/private)
- websocket.md: WebSocket catalog (public/private/common)
- fix.md: FIX catalog (official-page check result)
- data.md: Data distribution catalog (official-page check result)
- diffs.md: Official API diffs (changelog-only)
- catalog.json: Machine-readable SSOT (aligned with markdown)
- templates.md: Update templates
- CHANGELOG.md: Append-only change history
