# Upbit Official Docs Catalog (SSOT)

This directory contains an **official-docs-only** catalog for Upbit:
- REST: Quotation API / Exchange API (as documented)
- WebSocket: documented public/private channels + protocol details
- Official diffs from Upbit changelog (official only)

Completeness evidence and crawl roots are tracked in `sources.md`.

## Files
- `sources.md`: SSOT evidence of official coverage paths
- `rest-api.md`: REST endpoints catalog (quotation/exchange)
- `websocket.md`: WebSocket catalog (public/private/common)
- `fix.md`: FIX catalog (officially not applicable unless documentation appears)
- `data.md`: Official data distribution catalog (currently not applicable)
- `diffs.md`: Official API diffs (changelog-derived)
- `catalog.json`: Machine-readable SSOT (aligned with markdown)
- `templates.md`: Update templates for periodic refresh
- `CHANGELOG.md`: Append-only updates for this folder
