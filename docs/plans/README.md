# docs/plans（計画 / 可変）

このディレクトリは **Plan（計画）** を管理します。  
Core Spec（固定仕様）は変えずに、「何をいつどこまでやるか」「順序」「マイルストーン」をここに置きます。

## ルール
- Planは変わってよい（変わる前提）
- ただし、Planの実行は常に SSOT（contracts/specs）に整合すること
- 進捗の証明は Gate（CI/Runtime/Daily）と Integrity Report を基準にする
