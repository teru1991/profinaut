# docs/legacy/ (Non-Canonical References)
`docs/legacy/**` は **参照用の過去資料**です。ここにある内容は **正（SSOT）ではありません**。

## ルール
- SSOT（contracts/specs）と矛盾したら SSOT が正
- legacy しか情報が無い場合、採用したい内容は必ず SSOT 側へ移植する
- legacy を更新して “正” を増やさない（SSOT分裂防止）
