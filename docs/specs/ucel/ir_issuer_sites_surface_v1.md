# IR Issuer Sites Surface v1

This spec defines canonical issuer-site sources on UCEL IR surface.

Supported source IDs:
- `jp_issuer_ir_html_public`
- `jp_issuer_ir_feed_public`
- `us_issuer_ir_html_public`
- `us_issuer_ir_feed_public`

Each source must expose canonical issuer resolution, document list/detail, artifact list/download, and provenance metadata.
