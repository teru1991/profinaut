# IR Canonical Model

Core enums/types implemented in `ucel-core/src/ir.rs`:
- `IrSourceFamily`
- `IrSourceKind`
- `IrAccessPolicyClass`
- `IrAccessPattern`
- `IrIssuerIdentityKind`
- `IrDocumentFamily`
- `IrArtifactKind`

Canonical descriptors:
- `IrSourceDescriptor`
- `IrIssuerKey` / `IrIssuerAlias`
- `IrDocumentDescriptor`
- `IrArtifactDescriptor`

This model is the single typed target for 014C-014F source adapters.
