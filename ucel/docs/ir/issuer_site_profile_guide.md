# Issuer Site Profile Guide

Profile object includes:
- root URL, IR candidates, feed descriptors
- selector rules by section kind
- attachment rules (extensions/content types)
- language hints, market, issuer key binding
- max depth and page budget

Section kinds:
- IrTop
- NewsArchive
- FilingArchive
- PresentationLibrary
- FinancialResults
- SustainabilityLibrary
- GovernanceLibrary
- MiscLibrary
